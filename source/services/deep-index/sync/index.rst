Sync
====

The Kenshi Deep Index Sync oracle allows retrieving, storing and indexing
desired blockchain data into the Kenshi database clusters. This data can later
be queried or consumed using either the Kenshi Deep Index endpoint for GraphQL
or the Kenshi Reverse-API service.
